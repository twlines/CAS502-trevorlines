# Installation

## Prerequisites

- Python 3.10+
- pip

## Setup
```bash
git clone https://github.com/twlines/CAS502-trevorlines-dericktangap.git
cd CAS502-trevorlines-dericktangap
python -m venv venv
source venv/bin/activate # Windows: venv\Scripts\activate
pip install -r requirements.txt
```

## Data

PSED I and II files are included in the `data/` directory
